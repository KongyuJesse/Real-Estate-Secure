# REALESTATE-SECURE (Flutter)

This folder contains the Flutter setup for the REALESTATE-SECURE Android app.

## Run (Android)
1. Open a terminal in this folder.
2. Install dependencies:
   - `flutter pub get`
3. Run on a connected device or emulator:
   - `flutter run`

## Build APK
- `flutter build apk`

## Build App Bundle (Play Store)
- `flutter build appbundle`
