# Real-Estate-Secure-Project

This folder is reserved for the real estate project files inside the monorepo.

## Collaboration

For changes in this folder, use a short-lived branch from `main`, for example `docs/real-estate-secure-project-readme-update` or `feat/real-estate-secure-project-setup`.

Before opening a pull request:

- keep the change scoped to `Real-Estate-Secure-Project/` whenever possible
- document the purpose of any new files you add
- open the pull request into `main`
